Processor

The Processor or rather Microprocessor of the Nintendo Virtual Boy was for its time a complex type, having to run off batteries the processor had to be able to run with little power while processing games for a long duration.

The Processor used was a NEC V810 and was the first processor that was commercially available in Japan!

Here are some technical specifications:

  • 32-Bit RISC Processor at 0.02 GHz clock speed
  • 1 Core
  • 1 Megabyte of Dynamic Random Access Memory
  • 1 Kilobyte Cache

Compared with a modern day XBox One Processor:

  • 64 Bit AMD Jaguar 1.8GHz
  • 8 Cores
  • 8000 Megabytes of Dynamic Random Access Memory
  • 2048 Kilobyte Cache

As you can see this is quite old. But not bad for a game console made in 1995!

Screen

The screen came with the Virtual Boy as it was built into the console. Imagine an PS5 having its own personal screen!

However the screen was not like a 24 inch HD TV, more like a 2000 Nokia Brick. Lets compare with a common monitor used on computers:

Nintendo Virtual Boy:

  • Reflection Technology Inc SLA dual mirror-scan, high resolution LED displays
  • Resolution: 384 x 224 pixels
  • 50.2 Hz Horizontal Scan Rate
  • 4 colours with 32 levels of intensity

Dell Monitor (Modern):

  • IPS LED-backlit LCD monitor
  • Resolution: 1920 x 1080 pixels
  • 60 Hz Horizontal Scan Rate
  • 16.7 Million colours
